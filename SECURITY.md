# Security Policy

## Reporting a Vulnerability

We take the security of GAIA and GAIA-Agents seriously. If you discover a security vulnerability within this repository, please follow the process outlined in the [Main GAIA Security Policy](https://github.com/vishalsdk14/GAIA/blob/main/SECURITY.md).

Please **do not** open a public GitHub issue for security vulnerabilities. Instead, follow the coordinated disclosure process described in the link above.

### Supported Versions

Only the latest version of the agents and templates in this repository are currently supported for security updates.

### Response Time

The maintainers will acknowledge receipt of your report within 48 hours and provide a timeline for resolution.
